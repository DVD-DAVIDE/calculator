- [x] Addition
- [x] Subtraction
- [x] Multiplication
- [x] Division
- [x] Floating Point Output
- [ ] Floating Point Input
    - [x] Input
    - [ ] Input Validation (double decimals, etc.)
- [ ] Power
- [ ] Square Root
- [ ] n-th Root
- [ ] Factorials
